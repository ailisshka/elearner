Тема: Разработка интерактивной платформы для изучения иностранных языков
Данный репозиторий содержит исходный код курсового проекта, представляющего собой полнофункциональное веб-приложение (Full-stack) для автоматизации процесса обучения.

1. Технический стек
При разработке использована современная архитектура разделения интерфейса и серверной логики:

Frontend: React.js (библиотека для создания пользовательских интерфейсов).

Сборщик: Vite (инструментарий для быстрой разработки).

Backend: Node.js с использованием фреймворка Express.

База данных: MongoDB (NoSQL решение для хранения данных пользователей и учебного контента).

Управление версиями: Git.

2. Функциональные возможности
Программный комплекс реализует следующие модули:

Модуль аутентификации: Регистрация и авторизация пользователей с использованием защищенных протоколов (JWT).

Интерактивный тренажер: Система цифровых карточек (Flashcards) для эффективного запоминания лексики.

Образовательный контент: Структурированные уроки с системой проверки знаний (тестирование).

Личный кабинет: Визуализация учебного прогресса и управление персональным словарем.
